Description

Nassar is a beautiful easy to use Portfolio template built on bootstrap The template is responsive, so you can view it also in the mobile/tablets devices and it looks very nice in those devices. 

 

The template have :

  4. Unique design
  5. and much more...

Loading Screen

Turn Off Loading Screen

Open the html file using any text editor. We recommend Sublime Text. Now look for the following code and remove them.


    <!--Start Preloader-->

    <div class="page-loader startLoad">

      <div class="loader startLoad"></div>

    </div>

    <!--End Preloader-->

Save the file and refresh your browser.

Google Map

To change the google map to your location open the custom.js file and look for the following code and replace highlighted text with your own google map code and save the file.

    if ( $(contact_map).length ){


        googleMap("map", 40.712784, -74.005941 );

        $(contact_map).show();


    }

To get Latitude & Longitude please visit http://www.latlong.net/

contact form

To turn on contact form open file send.php then look for the following code and replace email adress to your email and you can also change message subject and save the file.
 
$recipient = "me@mail.com";// Change the recipient email adress to your adrees
$sujet =  "Sender information";// This is subject you can change it
